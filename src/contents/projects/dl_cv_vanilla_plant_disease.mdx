## Backstory Why I Made This Project? (Problem)

My hope is this project can help agriculture industries specifically for vanilla plant farmers to classify vanilla plant. This model use dataset from kaggle: [Vanilla Plant Diseases Dataset](https://www.kaggle.com/datasets/mihsanpermana/vanilla-plant-disease-image-dataset) with ***12k file of images*** including data original + augmentation data. This model is Computer Vision with type image classification, I built the model for learning `4 categories` of vanilla plant diseases, including following:
- Healthy
- Black Spot (Anthracnose)
- Red Spot
- Stem Rot

![Ilustration vanilla plant diseases, these photo taken from dataset](/assets_porto/dl_cv_vanilla_plant_disease/vanilla_classes.png)


## Goals Build This Project (Solution)

I made this Project for final assignment of online class about [Deep Learning (Computer Vision) at Dicoding Indonesia](https://www.dicoding.com/certificates/1OP8R1W4LZQK). This model is built using `Tensorflow` and `Keras` with `CNN (Convulational Neural Network)` and Transfer Learning (`MobileNetV2`)

Main goals for building this project, including the following: 

- My model I've built gonna be used for agriculture industries specifically vanilla plant farmers
- To exercise processing unstructure dataset (image files) being a model
- To exercise learning model with transfer learning (I used `MobileNetV2`)
- To exercise and exploring about Deep Learning (Computer Vision)
- To implement model AI into web application

## Performance Model I Have Built
With ***12k images dataset***, I used configuration data splitting with `70% train`, `10% validation`, and `20% test`.
