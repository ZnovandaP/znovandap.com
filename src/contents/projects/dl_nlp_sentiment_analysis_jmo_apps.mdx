## Backstory Why I Made This Project? (Problem)

My hope is that this project can help the company that built it to make improvements. This model use dataset from Google Playstore (I do the scraping data manually with my own script) with `100k+` reviews/threads. This model is NLP (*Natural Language Processing*) with type sentiment analysis, I built the model for learning `3 categories` of sentiment, including following:
- Positive
- Negative
- Neutral

## Goals Build This Project (Solution)

I made this Project for final assignment of online class about [Deep Learning (Natural Language Processing) at Dicoding Indonesia](https://www.dicoding.com/certificates/1OP8R1W4LZQK). This model is built using `Tensorflow` and `Keras` with `GRU (Gated Recurrent Unit)`

Main goals for building this project, including the following: 

- My model I've built gonna be used for analyze sentiment JMO (Jamsostek Mobile) on Google Playstore
- To exercise processing unstructure dataset (text files) being a model
- To exercise learning model with GRU (Gated Recurrent Unit)
- To exercise and exploring about Deep Learning (Natural Language Processing)
- To implement model AI into web application
