## Backstory Why Me Made This Project?

Project znovandap-portfolio v1 is a personal website, which contains personal my journey as front-end
web developer from zero. Project znovandap-portfolio v1 was build with 3 basic technology for build a 
website and some package dependencies. These include `HTML`, `CSS`, `Javascript`, `TailwindCSS`, 
and some supporting library like a library animation, etc.  

### Goals Build This Project

Main goals for building this project, including the following: 

- To exercise and increase fundamental 3 basic of technology for building website.
- Deeping `DOM Manipulation` with Javascript,
- Learning about how to styling website with CSS framework, on this project i'am use `TailwindCSS`
- Learning about use third party animatiaon library
- Learning about build interactive website.

I was learning a lot about front-end web development while build this project, after i finish this
project, i think my programming skills are improved over time.
